===== MMWAVE DFP Release 02.02.04.00 =====

TI mmWave Device Firmware Package (DFP) enables
the development of millimeter wave (mmWave)
radar applications using AWR2243 SOC/EVM.
It includes necessary components which will
facilitate end users to integrate AWR2243 SOC
with their choice of processor. It also includes
RF evaluation firmwares to help user to
measure RF and key system performance on
TI WaveSense Radar devices using mmWave Studio.
